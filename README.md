# Discord.js v14 Template

This is a simple Discord.js v14 template example. If you want to code a Discord bot using MongoDB, you can get a quick start with this setup. It fully supports slash commands, and all content is written in a way that everyone can understand. For installation or support, join the [Coders community](https://discord.gg/kyrSBzuepA).
