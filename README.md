# Discord.js v14 Bot Template

This is a high-performance, clean, and easy-to-use Discord.js v14 template. This setup provides a solid foundation for building any type of Discord bot. It fully supports slash commands and features a modular handler system for commands and events.

# Installation

1. Install **Node.js** on your computer.
2. Download this project as a ZIP file and extract it to a folder.
3. Configure the `src/config.json` file with your bot credentials.
4. Open a terminal (PowerShell or CMD) in the project directory.
5. Run the following command to install dependencies:
```
npm install
```
6. Start the bot by running:
```
node src/index.js
```
