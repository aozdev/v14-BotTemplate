This is a simple Discord.js v14 template example. If you want to code a Discord bot using MongoDB, you can get a quick start with this setup. It fully supports slash commands, and all content is written in a way that everyone can understand. For installation or support, join the [Coders community](https://discord.gg/kyrSBzuepA).

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
````

2. **Install dependencies:**

```bash
npm install
```

3. **Set up your `config.json`:**

Create a `config.json` file in the root directory and add your bot token and MongoDB URI like this:

```json
{
  "token": "BOT_TOKEN",
  "clientId": "CLIENT_ID",
  "developerId":""
}
```

> Replace `"your_discord_bot_token"` and `"your_mongodb_connection_string"` with your own credentials.

4. **Run the bot:**

```bash
node src/index.js
```

Your bot should now be online and ready to use!
